# 3D俄罗斯方块游戏

一个基于Cocos Creator的3D俄罗斯方块游戏项目。

## 项目结构

- `assets/scripts/` - 游戏逻辑脚本
  - `GameController.ts` - 游戏主控制器
  - `GridSystem3D.ts` - 3D网格管理系统
  - `BlockPiece3D.ts` - 3D方块组件
  - `GameUI.ts` - 游戏UI组件
  - `InputHandler.ts` - 输入处理器
  - `GameInitializer.ts` - 游戏场景初始化器
  - `Main.ts` - 项目入口脚本
  - `GameStateManager.ts` - 游戏状态管理器

## 开发环境设置

### 安装依赖

```bash
npm install
```

### 单元测试

本项目包含全面的单元测试，用于验证游戏核心逻辑。

```bash
# 运行所有测试
npm test

# 运行测试并监听更改
npm run test:watch

# 运行测试并生成覆盖率报告
npm run test:coverage
```

## 运行项目

1. 打开 Cocos Creator 3.8+
2. 导入项目
3. 确保 Main 场景被设置为启动场景
4. 运行场景

注意：如果场景中只有主灯光和主相机，请检查是否正确设置了场景初始化逻辑。

## 游戏特性

- 3D俄罗斯方块游戏玩法
- 多种方块形状
- 3D空间移动和旋转
- 完整的消除逻辑
- 得分系统
- **游戏状态保存与加载功能** - 可随时暂停和恢复游戏

## 游戏控制

### 移动控制
- **A键 / ←键**: 向左移动
- **D键 / →键**: 向右移动
- **W键 / ↑键**: 向前移动
- **↓键**: 向后移动
- **空格键**: 快速下落

### 旋转控制
- **Q键**: 绕Y轴顺时针旋转
- **E键**: 绕Y轴逆时针旋转
- **Z键**: 绕X轴旋转
- **C键**: 绕Z轴旋转

### 游戏控制
- **P键**: 暂停/继续游戏
- **S键**: 保存游戏进度
- **L键**: 加载游戏进度
- **R键**: 重新开始游戏

## 游戏状态管理

游戏支持状态保存和加载功能：

- 按"S"键可保存当前游戏进度
- 按"L"键可加载之前的进度
- 游戏状态包括：得分、等级、消行数、暂停状态、重力计时器、网格状态、当前方块姿态等
- 数据保存在浏览器的本地存储中
